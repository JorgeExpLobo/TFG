# AI COOKING APP – TFG PROJECT CONTEXT

Este proyecto es una aplicación que permite a un usuario:

1. Registrar ingredientes que tiene en casa
2. Guardar precios de compra
3. Generar recetas con IA
4. Crear dietas semanales
5. Analizar ingredientes disponibles

La lógica del backend se implementa usando n8n.

La base de datos es PostgreSQL.

La aplicación usa IA para:

- interpretar tickets
- normalizar ingredientes
- generar recetas
- interpretar precios
